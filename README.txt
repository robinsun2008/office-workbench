# 本地办公工作台 - 部署运行教程

## 项目简介

本地办公工作台是一个轻量级的Web应用，包含四大核心模块：
- **待办任务管理**：支持多条件检索、附件上传、进度跟踪
- **工作备忘录**：万字级长文本存储，自定义标签
- **重点关注看板**：可视化展示关注领域及事项状态
- **部门报告库**：存储各类部门文档，全文检索

技术栈：原生HTML+JavaScript前端、Python Flask后端、SQLite本地数据库

---

## 一、环境准备

### 1.1 安装Python

确保已安装Python 3.8或更高版本：

```bash
# Windows检查Python版本
python --version

# 如未安装，请从 https://www.python.org/downloads/ 下载安装
```

### 1.2 安装依赖

在项目目录下执行：

```bash
cd d:\mydoc\todolist
pip install -r requirements.txt
```

依赖包说明：
- Flask：Web框架
- Flask-CORS：跨域支持
- Werkzeug：文件上传处理
- python-dateutil：日期处理

---

## 二、启动应用

### 2.1 初始化数据库

首次运行需要初始化数据库：

```bash
python init_db.py
```

执行后会创建：
- `data/office.db` - SQLite数据库文件
- 默认配置数据（任务标签、团队、人员等）

### 2.2 启动Web服务

```bash
python app.py
```

启动成功后会显示：
```
==================================================
本地办公工作台启动中...
访问地址: http://localhost:5000
==================================================
```

### 2.3 访问应用

打开浏览器访问：http://localhost:5000

## 三、功能使用说明

### 3.1 待办任务模块

**核心功能：**
- 创建任务：点击"新建任务"按钮，填写摘要、截止时间、标签等
- 扩展字段：点击"详细信息"折叠区展开编辑详细内容、进度备注
- 附件上传：在任务详情区点击"上传附件"
- 多条件检索：支持按关键词、标签、团队、人员、状态筛选
- 任务完成：点击"完成"按钮标记任务为已完成

**首页提醒：**
- 自动展示未来3天内到期的待办任务

### 3.2 工作备忘录模块

**核心功能：**
- 创建备忘录：填写标题、内容（支持万字级）、自定义标签
- 附件上传：支持上传相关附件文件
- 检索：按关键词或标签筛选

### 3.3 重点关注看板模块

**核心功能：**
- 创建关注领域：点击"新建领域"添加关注分类
- 添加事项：在每个领域卡片内点击"+ 添加事项"
- 事项字段：标题（≤40字）、备注、计划时间、状态
- 状态流转：未开始 → 进行中 → 完成/延期
- 自动过滤：完成超过1个月的事项自动隐藏

### 3.4 部门报告库模块

**核心功能：**
- 创建报告：填写标题、分类、标签、内容
- 分类选项：自查报告、审计报告、工作周报、工作月报等
- 全文检索：支持按标题、内容、标签搜索
- 查看详情：点击"查看"按钮展开完整内容

### 3.5 系统配置模块

**可配置项：**
- 任务标签：紧急、重要、一般（可自定义添加）
- 指派团队：技术部、产品部等（可自定义）
- 指派人员：张三、李四等（可自定义）
- 关注领域：项目进度、团队建设等（可自定义）

## 四、数据导入导出

### 4.1 导出全部数据

在首页或配置页点击"导出全部数据"：
- 生成ZIP压缩包
- 包含所有业务数据（JSON格式）
- 包含所有附件文件（带标识命名）

导出文件命名规则：
- `office_export_YYYYMMDDHHMMSS.zip`

附件标识命名：
- `模块类型_记录ID_原始文件名`
- 例如：`todo_5_项目文档.pdf`

### 4.2 导入数据

点击"导入数据"按钮：
- 选择之前导出的ZIP文件
- 系统自动恢复所有数据和附件关联

**注意：** 导入会覆盖现有数据，请谨慎操作

### 4.3 仅导出附件

点击"导出附件"：
- 仅导出所有附件文件
- 包含附件信息JSON文件

## 五、附件存储说明

### 5.1 存储位置

附件统一存储在：
```
d:\mydoc\todolist\uploads\
├── todo\      # 待办任务附件
├── memo\      # 备忘录附件
└── report\    # 报告附件（预留）
```

### 5.2 支持的文件类型

允许上传的文件扩展名：
- 文档：txt, pdf, doc, docx, xls, xlsx
- 图片：png, jpg, jpeg, gif
- 压缩：zip, rar

### 5.3 文件大小限制

单文件最大：50MB

## 七、常见问题

### Q1: 启动报错"端口被占用"

修改 `app.py` 最后一行端口：
```python
app.run(host='0.0.0.0', port=5001, debug=True)
```

### Q2: 数据库初始化失败

确保 `data` 目录存在且有写入权限：
```bash
mkdir data
python init_db.py
```

### Q3: 附件上传失败

检查 `uploads` 目录权限，确保目录存在：
```bash
mkdir uploads
mkdir uploads\todo
mkdir uploads\memo
```

### Q4: 如何备份数据

两种方式：
1. 使用"导出全部数据"功能导出ZIP文件
2. 直接复制 `data/office.db` 和 `uploads/` 目录
